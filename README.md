# Hız İhlali Tespit Projesi

Bu proje, C programlama dersi kapsamında hazırlanmıştır.

## Proje Amacı
Ankara giriş ve Gölbaşı çıkış noktalarındaki araçların
ortalama hızlarını hesaplayarak hız sınırını aşan araçları tespit etmek.

## Kullanılan Yapılar
- Struct
- Dosya işlemleri (FILE)
- String işlemleri
- Zaman hesaplama

## Klasör Yapısı
- src/  → C kaynak kodları
- data/ → Giriş / çıkış verileri

## Derleme
```bash
gcc src/main.c -o hiz
